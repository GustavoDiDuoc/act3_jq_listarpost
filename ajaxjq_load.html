El metodo load() de JQuery es un metodo de JQuery es bastante simple y bien poderoso. Nos sirve para cargar los datos de un servidor y coloca los datos devuelto en algun elemento seleccionado
la sintaxis es:
$(selector).load(URL,data,callback);
Los parametros:
URL especifica lo que se desea cargar (API).
Data especifica un conjunto de datos clave y valor de consulta para ser enviado junto con la solicitud.
callback es la funcion que se ejecutar despues de load cuando se complete el metodo.

este elemento debe cambiar cuando obtenga el dato
